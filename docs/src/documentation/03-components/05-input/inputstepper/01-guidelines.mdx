## When to use

- To let users change small amounts of something and submit it.
- For data such as the number of passengers on a trip.

## When not to use

- To change values not in a form---use a [stepper](/components/interaction/stepper).
- For value that could change by a lot or irregularly---use an [input field](/components/input/inputfield)
  set to accept numbers.
