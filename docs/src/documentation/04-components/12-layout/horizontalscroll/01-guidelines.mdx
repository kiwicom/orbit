## When to use

- To present similar items (such as [cards](/components/structure/card/) with baggage options) as a single row even on small screens.

### When not to use

- To add spacing to elements---use a [stack](/components/layout/stack/).
- To present items that can span multiple rows---use a flexible [box](/components/layout/box/).
