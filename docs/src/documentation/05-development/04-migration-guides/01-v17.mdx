# Orbit Migration Guide v17

This migration guide focuses on the process of migrating from Orbit v16 to v17.0, as some breaking changes were introduced.
With this guide, we aim to walk through all the breaking changes and how they can be addressed, allowing the migration to be smoother and effortlessly.

## Breaking changes

### ColoredTwitter and Twitter icons removed

The `ColoredTwitter` and `Twitter` icons were removed. The `X` icon should be used instead.

## Deprecations

### `Twitter` icon

The `Twitter` icon was deprecated and will be removed in a future version. The `X` icon should be used instead.
