import defaultTheme from "@kiwicom/orbit-components/lib/defaultTheme";

You can set two things about illustrations.
The first is the illustration itself,
which you can choose from the [complete list of illustrations](#available-illustrations).

The second is the size.
We have five different sizes to offer, which are based around the maximum height of the illustration.

<ul>
  <li>Extra small ({defaultTheme.orbit.illustrationHeightExtraSmall})</li>
  <li>Small ({defaultTheme.orbit.illustrationHeightSmall})</li>
  <li>Medium ({defaultTheme.orbit.illustrationHeightMedium})</li>
  <li>Large ({defaultTheme.orbit.illustrationHeightLarge})</li>
  <li>Display ({defaultTheme.orbit.illustrationHeightDisplay})</li>
</ul>
